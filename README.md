# FunChat Frontend 2026

React + Vite user app for random chat/video.

## Structure
- `src/components` UI components
- `src/hooks` socket + WebRTC hooks
- `src/lib` constants/config
- `src/styles` app styles

## Run
```
cd funchat_frontend_2026
npm install
npm run dev
```

## Env
Optional:
```
VITE_BACKEND_URL=http://localhost:4000
```
